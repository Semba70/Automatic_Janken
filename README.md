# Automatic_Janken

### じゃんけん画像の自動判別

手の画像を入力とし、自動で手の座標とグーチョキパーの判定を行う

<img src="./sample1.png" width="500px">

上のような画像の場合

-----------------------------

$ ./a.exe sample1.ppm

75 251 310 134 t l

512 128 175 559 g w

-----------------------------

矩形の座標、手の形、勝敗を出力する

g = グー、t = チョキ、 p = パー

w = 勝者、 w = 敗者
